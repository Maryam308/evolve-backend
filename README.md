# Evolve - Backend
## Description
This is the backend API for **Evolve**, a personal growth web application that helps users document their achievements and reflect on their self-development journey.
## Front-End Repository
For full details about the project concept, features, UI, and usage, please check the front-end repository below:
**Front-End Repository:** https://github.com/Maryam308/evolve-frontend
## Technologies Used
- **Node.js** - Runtime environment
- **Express.js** - Web application framework
- **MongoDB** - NoSQL database
- **Mongoose** - MongoDB object modeling
- **JWT (JSON Web Tokens)** - Authentication and authorization
- **bcrypt** - Password hashing
